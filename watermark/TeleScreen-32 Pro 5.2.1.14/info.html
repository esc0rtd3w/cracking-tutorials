    From DRS-Digitrax :

    This program scrolls text on screen, much in the way a teleprompter does. Features include the use of Windows fonts and colors, reverse mirror text, color bars, and alt files. The program also offers countdown generators with TV-style titles and variable scroll speed and font size. This update features full antialias text, drag-and-drop scroll window sizing, improved video titling (fading), and an improved online help system.
