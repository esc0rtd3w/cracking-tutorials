    CNET editors' review

    Reviewed by: CNET Staff

    Children will enjoy using this program to unscramble puzzles but they'll need help to get started. Visual Shapes was designed to boost children's creative skills by helping to distinguish shapes and objects through puzzle making. Its colorful interface is aesthetically pleasing. Before getting started, users will need to click the much-needed Give Me A Hint! button to see the finished puzzle. To move a tile, simply drag and drop it into the correct location. You can upload your own images and make your own jigsaw puzzles. However, you can only upload images sized 400 x 400 pixels. Fortunately, the program has plenty of different high-resolution visual shapes to choose from, including stars, triangles, and cylinders. This program, limited to 15 uses in the trial, is recommended for use by preschoolers and primary school students.

    Publisher's description

    From Caltrox Educational Software :

    Visual Shapes is a unique logic puzzles method of teaching young children recognition of common visual shapes and objects using a jigsaw puzzles assembling format. Visual Shapes has been designed for to be easy to use as a puzzle maker to create an unlimited variety of fun tests and creative ability training for children. Full audio-visual and multimedia features have been built into the jigsaw puzzles application. A unique feature is that you can also load your own images into the program in JPEG, GIF and BMP formats to create your own custom jigsaw puzzles, as well as use the over 60 high-resolution graphics included in the program. Features include over 60 different visual shapes and objects to work on, inbuilt realistic puzzle maker with high-resolution graphics, load your own pictures in JPEG, GIF, and BMP formats, fully multimedia based fun tests with audio-visual output. Version 4 includes unspecified updates.
