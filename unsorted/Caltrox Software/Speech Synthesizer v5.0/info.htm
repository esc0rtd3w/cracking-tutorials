    CNET editors' review

    Reviewed by: CNET Staff

    This text-to-speech program can help serve as a language-learning aid, though we wouldn't call its speech prowess impressive. Speech Synthesizer can read TXT, DOC, or RTF files, and it also can recite any text you enter into the box. The application has a pleasing graphical interface that offers several nice features, including a tool for recording the speech and an animated clown that mouths the words. The program has nearly 20 voice outputs, and you can modify the speed, voice tone, and pitch. Unfortunately, the program uses the Microsoft Speech Engine, which often stumbles on words and can be hard to understand. Also, users unfamiliar with written English will wish the program highlighted or scrolled through the words as the program is speaking. The 15-use trial period could be more generous. Speech Synthesizer isn't a perfect teaching tool, but if you're looking to brush up on your English skills, you might give it a try.

    Publisher's description

    From Caltrox Educational Software :

    Speech Synthesizer is a great text-to-speech synthesizer for children to learn English speaking and vocabulary. Speech Synthesizer is a powerful text-to-voice converting application that is extremely useful for converting all types of text to speech, learning English pronunciation, and vocabulary building. The TTS software runs on all Windows systems and has adjustable speaking speeds, as well as an aesthetically designed user interface.

    Its features include a realistic embedded multiplex text-to-speech synthesizer, reads all text, RTF, and Word documents, customizable display area, animated speaking character, eighteen voice outputs, adjustable speaking speeds, variable voice tones from low to high pitch, graphical sound output visual display, and you can record the speech output as WAV format as well. The software is designed specifically for kids with a fully graphical Windows-based interface. Version 5 has minor updates and bugfixes.
