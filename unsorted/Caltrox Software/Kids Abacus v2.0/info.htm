    CNET editors' review

    Reviewed by: CNET Staff

    This virtual abacus will help young children learn rudimentary counting skills. Tykes will like Kids Abacus s pleasing, colorful interface, which lets them learn numbers one to 100 using balls, bunnies, butterflies, boxes, and cakes. Kids simply drag objects from the main abacus to a small circle to find out how many items they ve counted and how many remain. The voice that counts along with each action makes learning easier, but we d still like an option for turning it off. Though the program is clearly only intended for kids in the preschool age bracket, it is totally free and perfectly functional at what it does.

    Publisher's description

    From Caltrox Educational Software :

    This is free, worksheet-based learning software that uses a graphical method of learning numbers and counting from one to 100. The software has been designed for ease of use, fast setup, and interactivity. Full audiovisual and multimedia features have been built into the program to make learning the basics of numbers and counting an enjoyable experience for children. It changes the usual difficult and tedious learning experience with math and makes the subject easy to understand using a logical step-by-step process that is both interesting and fun to use for the average school-going child.

    The following are the features of Kid's Abacus 2: the ability to count from one to 100 using graphic objects, five picture objects to count, a drag-and-drop method of counting, arealistic speech synthesizer to aid in counting. The application is fully multimedia based with audiovisual output. Version 2 has minor updates and bugfixes.
