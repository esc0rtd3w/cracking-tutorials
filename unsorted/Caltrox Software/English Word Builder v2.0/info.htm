    CNET editors' review

    Reviewed by: CNET Staff

    Anyone who's learned a second language knows that games and puzzles can help build vocabulary more quickly that simply studying lists. English Word Builder is a word puzzle that aims to develop players' spelling skills, as well as enlarge vocabulary. Unfortunately, this laudable goal is somewhat undermined by the program's spotty design. The goal is to use a group of five available letters to build English words, filling in fields before time runs out. Each field has a specific solution, so that even if another word might fit using the available letters, it gets rejected. This can be a bit confusing for new English learners, we fear. The game gives points for each right word, but some words can be reentered over and over again, with points added every time, until the program simply crashes. On a more basic level, the interface itself is fairly primitive, particularly compared to other commercial word games. None of these flaws are fatal, and the game could certainly be of help in building English vocabulary, but players will have to keep these quirks in mind.
